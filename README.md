# CodeAlpha Image Gallery

A simple and responsive Image Gallery web application built using **HTML, CSS, and JavaScript**.  
It allows users to view images in a clean grid layout with interactive features like preview/lightbox.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- 
##  Features
- Responsive image grid layout
- Click to view image in fullscreen (lightbox)
- Next / Previous image navigation (if added)
- Like / Favorite feature (if implemented)
- Smooth and simple UI design
## How to Run Locally
1. Download or clone the repository  
2. Open `index.html` in browser  
3. Or use VS Code Live Server extension
